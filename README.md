# 驾校管理系统 java394

## 项目概述

驾校管理系统是一款集成前台和后台功能的全面管理系统。该系统主要服务于驾校学员、教练以及管理员，提供车辆信息管理、考试安排、在线学习、预约管理以及公告浏览等多样化功能，满足不同角色的使用需求。

## 技术栈

- 后端：Springboot、Spring、SpringMVC、Mybatis
- 前端：HTML、CSS、JavaScript、Vue2

## 运行环境

- JDK 1.8
- MySQL 5+
- IDEA
- Maven 3
- Node 12.21.3

## 功能模块

### 前台端

- **驾校车辆**：查看车辆信息，包括车型、车辆状态等。
- **考试安排**：查看即将进行的考试安排详情。
- **驾驶知识**：在线学习驾驶相关知识。
- **笔试考试**：在线预约笔试考试，并查看预约状态。
- **驾校公告**：浏览驾校发布的各类公告。

### 后台端

#### 面向学员

- **驾校车辆**：查看车辆详情，了解训练车辆情况。
- **考试安排**：查看考试安排，提前准备。
- **驾驶知识**：学习交通法规、驾驶技巧等。
- **笔试考试**：在线预约，查看预约状态。
- **驾校公告**：获取驾校最新动态。

#### 面向教练

- **教练信息管理**：编辑个人信息。
- **学员管理**：查看学员信息，跟踪学习进度。
- **教学任务管理**：管理教学计划。
- **考试预约管理**：协助学员预约考试。
- **成绩录入**：记录学员成绩，生成报告。

#### 面向管理员

- **用户管理**：管理所有用户信息。
- **驾校教练管理**：审核教练信息。
- **驾校车辆管理**：分配车辆信息。
- **预约信息管理**：监控管理预约情况。
- **考试安排管理**：设定考试安排。
- **考试预约管理**：审核学员预约请求。
- **驾驶知识管理**：更新知识库内容。
- **学员成绩管理**：统计分析学员成绩。
- **笔试考试管理**：设置笔试试题库，管理考试流程。
- **考试题目管理**：维护题目库。
- **系统管理**：进行系统设置，保障系统稳定运行。

## 系统角色

- 学员
- 教练
- 管理员
